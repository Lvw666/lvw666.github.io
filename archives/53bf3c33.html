JS 数据类型

发表于2022-05-22|更新于2022-05-22|Web 前端☛ JavaScript✦ ECMAScript
|字数总计:4.1k|阅读时长:18分钟|阅读量:

什么是数据类型

数据类型就是数据的类型

为了方便计算和操作数据, 我们把数据分了不同的类型, 不同编程语言数据类型略有不同

变量是存放数据的容器, 变量和数据的关系, 相当于瓶子和液体的关系

强类型: 油瓶只能装油, 醋瓶只能装醋 Java

弱类型: 装油就是油瓶, 装醋就是醋瓶 JavaScript

JS的数据类型

判断数据类型

使用 typeof 运算符, 可以检测值或者变量的类型

基本数据类型的检测结果

温馨提示: typeof 打印出来的信息, 它的数据类型是字符串

JS是弱类型语言

JS 是弱类型语言, 变量的数据类型可以发生改变

prompt 函数

alert 负责弹框, 用户点击确定可以关闭弹框

prompt 也是弹框, 但是可以接收用户的输入, 用户输入完成后, 点击确定关闭弹框

注意: 用户信息的数据类型是 string, 如果用户点击取消, 返回 null

undefined 类型

一个没有赋值的变量, 默认值是 undefined

undefined 的类型, 也是 undefined

undefined 既是值, 又是数据类型, 这种类型只是有它自己一个值

注意: undefined 不加引号

null 类型

null 表示空, 它是空对象

使用 typeof 检查 null 的类型, 结果是 object

null 和 undefined 的区别

null 会显式赋值, undefined 往往不会

空字符串和空对象不一样

Number 数字类型

所有数字, 不分大小, 不分整浮, 不分正负, 都是数字类型

数字 0 可以省略

0.6, 可以写成 .6

科学数字法

较大数或者较小数(绝对值较小), 可以写成科学计数法

不同进制的数字

js 中用到的进制, 包括十进制, 二进制, 八进制, 十六进制

二进制0b开头, 八进制0开头, 十六进制0x开头

console.log() 输出的数字式十进制的

NaN

not a number ==> NaN (区分大小写)

NaN 属性是代表非数字值的特殊值, 不是数据类型之一

数学运算中, 如果无法返回一个数字, 往往会返回 NaN

注意: NaN 不是一个数字, 但是是一个数字类型的值

注意: 虽然都是 NaN, 但是是不相等的

注意: NaN 可以参与运算, 返回结果都是 NaN

string 字符串类型

字符串就是人类的自然语言

字符串要用引号包裹, 单引号双引号都可以, 但是要成对

数字两边加上引号, 也是字符串, 它只是看上去像数字而已

比如手机号, 身份证号, 学号, 虽然是由数字组成, 但是不牵涉数学运算, 应该做成字符串类型

字符串拼接

使用+, 可以拼接字符串

如果两边是数字, 表示数字相加

如果两边, 有一边是字符串, 表示字符串拼接

如果一边是数字, 另一边是空字符串, 结果是数字, 但数字类型是String

思考运行结果

字符串和变量拼接

字符串拼接时, 可以插入变量

来一个复杂一点的

字符串的换行

字符串换行, 左右两个引号不能出现在不同的行里面, 不然报错

可以使用\n换行, 相当于html里面的br标签

空字符串

空字符串, 表示字符串是空的, 里面没有内容, 用空的引号对表示

空字符串可以和数字拼接, 结果为数字, 但是会将数字转为 String 类型

字符串的长度

使用length属性, 可以获取字符串的长度

length ==> 长度

点是的的意思

str.length, str 的长度, 类似于张三 的 年龄 zhangsan.age

字符串的常用方法

方法, 类似于功能, 通过方法, 可以让字符串做一些事情

类似于于谦.抽烟(),于谦.喝酒(),于谦.烫头()

charAt (指定位置)

可以得到指定位置的字符, 叫号, 编程从 0 开始计数

传数字, 返回字符串

substring (范围)

substring(a,b), 得到的是从 a 开始, 到 b 结束(不包括 b)的子串

如果 b 超出了最大范围, 到结束为止

substring(a,b), 如果省略第二个参数, 返回的子串, 会一直到字符串的结尾

substring(a,b), a 可以大于 b, 数字顺序, 会自动调整为小的数字在前

slice (范围,可负开)

slice(a,b), 得到从 a 开始, 到 b 结束(不包括 b)的子串

和 substring(a,b), 非常像

substr (长度)

substr(a,b), 得到从 a 开始, 长度为 b 的子串, a 代表起始位置, b 代表长度

substr(a,b), b 可以省略, 表示到字符串的结尾

如果只有一个参数, substr()和 substring(), 作用相同

substr(a,b), a 可以是负数, 表示倒数的位置

substring 和 slice 的区别

slice(a,b)的参数 a, 可以是负数, 表示倒数

substring(a,b)的参数 a, 不可以是负数, 如果是负数, 会被当成 0

slice(a,b)中, a 的值必须小于 b 的值, 不然会找不到子串, substring(a,b), 会自动更换顺序

substring、substr、slice对比

substring(a,b) 和 slice(a,b) 功能基本一致, 都是得到从 a 开始, 到 b 结束, 不包括 b 的子字符串

  • substring(a,b), 可以自动交换两个参数的位置, slice(a,b)不可以
  • slice(a,b)的参数 a 可以是负数, substring(a,b)不行

substr(a,b)中, 参数 b 是子串的长度, 不是位置编号

toUpperCase, toLowerCase

toUpperCase(): 字符串转大写

toLowerCase(): 字符串转小写

indexOf

indexOf(), 返回某个指定子串, 在字符串中首次出现的位置

传字符串返回数字/索引

如果没有找到, 返回-1

借助 indexOf, 就可以判断字符串中是否包含某些内容, 过滤敏感词汇

布尔类型 boolean

注意:

  1. true 和 false 区分大小写, 只有全小写的状态才是对的
  2. 不加引号, 如果加引号, "true"表示字符串
  3. 单词一定要记住, 因为容易写错
  4. 关系和逻辑运算, 产生的结果就是布尔类型

数据类型转换

其他值转数字

使用 Number(), 可以把任意数据类型, 转成数字

纯数字字符串, 会变成数字, 不是纯数字的字符串, 会变成 NaN not a number

布尔值, true ==> 1, false ==> 0

undefined ==> NaN, null ==> 0

parseInt() 可以把字符串, 转成整数, 不会四舍五入

会自动截掉第一个非数字字符之后的所有字符

不是数字开头, 转成 NaN

parseFloat() 可以把字符串, 转成浮点数/小数

会自动截掉第一个非数字字符之后的所有字符

不是数字开头, 转成 NaN

其他值转字符串

String(), 可以把任意数据类型, 转成字符串

会把值, 转成”长得相同”的字符串

非十进制或者科学计数法会转成十进制

toString(), 所有的值都有这个方法, 作用是把当前值转成字符串

效果和规则, 与 String()相同

其他值转布尔值

Boolean() 可以把任意数据类型, 转成布尔值

数字转布尔值, 0 和 NaN 会变成 false, 其他都是 true

字符串转布尔值, 空字符串会变成 false, 其他的都变成 true

undefined 和 null 转布尔值, 都会转成 false
