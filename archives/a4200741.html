Composition API

Composition 构成, 组成

随着代码量越来越大, 代码的可维护性会降低, Composition API 可以提高代码的可维护性

其中有一个比较重要的函数, setup 函数, 它是在实例被完全初始化之前会自动执行

setup 导出的内容可以直接在模板中使用

也可以使用 setup 返回的方法

在 setup 里面不能使用 methods 里面定义的方法

原因是 this 还没有绑定 app 实例, 所以无法调用 this.showHello

setup 函数, 它是在实例被完全初始化之前会执行, 所以通常不会在 setup 中使用 this

但是可以在 methods 里面使用 setup 返回的方法, 不过要用$options

ref

Vue 是数据驱动, 并不推荐直接操作 DOM, 但是不代表完全不能获取 DOM 元素

可以使用原生的方式, 或者使用 ref, 都可以获取 DOM 元素

ref 是 reference 的前三个字母

组件也可以使用引用

通过 ref, 可以在父组件中, 触发子组件的函数

ref 是获取 DOM 节点/组件引用的一个语法
