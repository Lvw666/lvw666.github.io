定时器和延时器

发表于2022-06-22|更新于2022-06-22|Web 前端☛ JavaScript✦ DOM
|字数总计:3.6k|阅读时长:16分钟|阅读量:

定时器

setInterval()函数

setInterval()函数可以重复调用一个函数，在每次调用之间具有固定的时间间隔

setInterval()的参数

setInterval()函数可以接收第 3、4……个参数，它们将按顺序传入函数

具名函数也可以传入 setInterval

具名函数也可以传入 setInterval

清除定时器

clearInterval()函数可以清除一个定时器

延时器

setTimeout()函数

setTimeout()函数可以设置一个延时器，当指定时间到了之后，会执行函数一次，不再重复执行。

清除延时器

clearTimeout()函数可以清除延时器，和 clearInterval()非常类似

复杂一点的拆炸弹示例

异步语句

setInterval()和 setTimeout()是两个异步语句

异步（asynchronous）：不会阻塞 CPU 继续执行其他语句，当异步完成时，会执行“回调函数”（callback）

同步: 第一步执行完, 开始执行第二步, 第二步执行完, 开始执行第三步…

异步: 第一步开始执行, 继续执行第二步, 继续执行第三步, 第一步执行完了, 执行回调函数

Tips: 回调函数: 函数能够当作参数进行调用

永远都是同步先执行, 再执行异步(因为默认把所有异步语句都放入一个异步队列中)

使用定时器实现动画

使用定时器可以实现动画，利用的就是“视觉暂留”原理

使用定时器实现动画较为不便：

  1. 不方便根据动画总时间计算步长
  2. 运动方向要设置正负
  3. 多种运动进行叠加较为困难（比如一个方形一边移动一边变为圆形）

CSS 的过渡属性

CSS 过渡允许在给定的时间内平滑地改变属性值

语法结构

如需创建过渡效果，必须明确两件事：

  1. 要添加效果的 CSS 属性
  2. 效果的持续时间

属性+过渡时间

可以监听一个属性

也可以监听两个属性

速率曲线

第三个参数是速率曲线

transition-timing-function 属性可接受以下值：

  • ease - 规定过渡效果，先缓慢地开始，然后加速，然后缓慢地结束（默认）
  • linear - 规定从开始到结束具有相同速度的过渡效果
  • ease-in -规定缓慢开始的过渡效果
  • ease-out - 规定缓慢结束的过渡效果
  • ease-in-out - 规定开始和结束较慢的过渡效果
  • cubic-bezier(n,n,n,n) - 允许您在三次贝塞尔函数中定义自己的值

延迟时间

第四个参数是延迟时间

transition-delay 属性规定过渡效果的延迟（以秒计）

和 border 类似, transition 支持连写

属性 描述
transition 简写属性，用于将四个过渡属性设置为单一属性。
transition-delay 规定过渡效果的延迟（以秒计）。
transition-duration 规定过渡效果要持续多少秒或毫秒。
transition-property 规定过渡效果所针对的 CSS 属性的名称。
transition-timing-function 规定过渡效果的速度曲线。

JS 和 CSS3 配合来实现动画

CSS3 的 transition 过渡属性可以实现动画

JavaScript 可以利用 CSS3 的 transition 属性轻松实现元素动画

JS 确定起止点, CSS3 过渡, 负责中间的动画效果

JS 和 CSS3 结合实现动画规避了定时器制作动画的缺点

示例:

  1. 点击按钮, div 从左(200px)跑到右边(1200px)
  2. 如果跑到右边了, 再次点击按钮, 反向
  3. 在运动的过程中, 点击可以反向(200px 结束)

函数节流

函数节流：一个函数执行一次后，只有大于设定的执行周期后才允许执行第二次

函数节流非常容易实现，只需要借助 setTimeout()延时器
