Vue 组件基础

组件

组件: 页面上的一部分, 可以把一个大的网页, 拆分成很多个小组件进行维护

根据页面来拆分, 组件化开发

根据逻辑来拆分, 模块化开发

最小的组件就是单个标签

使用组件需要三步

  1. 创建应用, 在应用中使用组件
  2. 定义组件
  3. 挂载到 DOM 元素

组件中, 也可以包含自己的样式和数据

使用组件时, 也可以进行传值

根组件

使用组件需要三步

  1. 创建应用, 在应用中使用组件, const app = Vue.createApp()
  2. 定义组件 app.component()
  3. 挂载到 DOM 元素 app.mount()

app.mount()的返回值, 就是根组件

组件的定义及复用

Vue.createApp({})在创建应用时, 接收一个对象作为参数, 这个对象就是根组件的一些设置

可以通过 app.component 来定义子组件

语法结构如下：

组件之间相互独立, 而且可以复用

全局组件和局部组件

全局组件

可以通过 app.component 来定义子组件, 定义的是全局组件, 不只是在根组件, 在其他组件中, 也可以直接使用

全局组件的优点是, 定义之后, 在其他组件中可以直接使用, 使用简单, 但是缺点是一直挂在 Vue 应用上面, 会有一定的性能损耗

局部组件

局部组件, 跟全局组件相对, 不损耗性能, 但是使用起来有一点麻烦

  1. 声明局部组件的代码要写在使用局部组件的前面
  2. 使用局部组件时, 需要先在 components 属性中声明

注意: 所谓声明, 其实就是做一个变量名和标签名的映射, {counter} 相当于{counter:counter}

局部组件的命名

使用局部组件时, 因为本质是定义变量, 所以需要遵守 JS 变量的命名规范, 我们可以使用大驼峰的命名方式, Vue 会自动把大驼峰和中划线的命名进行互转, 全局组件无此限制, 可以直接使用中划线的命名方式
