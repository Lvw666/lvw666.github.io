v-on

这个指令用于监听 DOM 事件，如点击 (click)，输入 (input) 等事件。

例如：实现字符串翻转效果

v-on:click="函数名" 可以简写成 @click="函数名"

v-model

条件渲染

相关指令有 v-if 和 v-show

控制 dom 元素的存在与否, 来控制显示和隐藏

v-if

if 判断, 如果为 true, 标签展示, 如果为 false, 标签不展示

所谓不展示, 就是连对应的 html 代码都没有, 相当于原来的 html 代码就没写

v-if 有配套的 v-else

v-if 还有配套的 v-else-if

v-show

v-show 和 v-if 类似, 通过结果是 true 或者 false, 来决定 DOM 元素的显示和隐藏

不过 v-show 通过 display:none 来隐藏 dom 元素, 所以牵涉到频繁的切换显示和隐藏状态的话, 推荐使用 v-show

如果数据不是一个数组, 而是一个对象, 推荐使用 for-in

遍历数组时推荐使用 of，语法格式为(item, index)

  • item: 迭代时不同的数组元素的值
  • index: 当前元素的索引

循环遍历对象时推荐使用 in，语法格式为(item, name, index)

  • item: 迭代时对象的键值
  • name: 迭代时对象的键名
  • index: 当前元素的索引

v-for 为了节约性能, 需要搭配 key 来使用, key 值要唯一

实际开发过程中, 通常使用数据中的 id 作为 key 的值

v-for 可以循环数字, for-in, for-of 都支持
