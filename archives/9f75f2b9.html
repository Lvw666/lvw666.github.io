表格标签

什么是表格

table, tr, td 标签

table 的 border 属性

为了让表格能够显示边框，table 标签通常有 border 属性

table 的 caption 属性

caption 是表格的标题， 它常常作为 table 的第一个子元素出现

th 标签

<th> 是标题小格，可以替代 <td>，表示一列的标题

单元格的宽高

和 img 标签类似, table 的宽高, 可以使用 width 和 height 来设置, 也是不用加单位

单元格的合并

什么是单元格合并

colspan 属性

colspan 属性用来设置 td 或者 th 的列跨度

rowspan 属性

rowspan 属性用来设置 td 或者 th 的行跨度

同时有 rowspan、colspan 属性

cellspacing 属性

cellspacing 设置单元格之间的距离, 如果值为 0, 代表单元格之间没有缝隙

cellpadding 属性

cellpadding 定义了内容和单元格边框之间的距离

表格的其他特性

thead、tbody、tfoot 标签

thead 标签定义表头
tbody 标签定义表核心内容
tfoot 标签定义表脚，通常是汇总行
