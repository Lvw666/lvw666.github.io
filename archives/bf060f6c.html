Vue 框架

Vue 是一套用于构建用户界面的渐进式框架

  1. Vue 是用于开发界面的
  2. 渐进式的框架, 是从使用 vue 的方式上面来进行的定义的, 即: 声明式渲染->组件化应用->客户端路由->集中式状态管理->项目构建

作者是尤雨溪

MVVM 设计模式

M -> Model 数据层

V -> View 视图层

VM -> ViewModel 视图数据链接层

组件 .$data, 可以获取组件的数据, 能获取就能修改

代码解析

createApp()可以创建 Vue 应用, mount 可以把这个应用挂载到指定的 DOM 上

mount 里面跟的是 CSS 选择器, 通过 CSS 选择器, 找到你对应的 DOM 元素

template 的意思是模板, 右边是模板对应的 HTML 代码, 可以把 HTML 代码展示在指定的 DOM 元素里面, 相当于 DOM.innerHTML = 模板内容

createApp 表示创建一个 Vue 应用, 里面传入一个对象作为参数, 这个对象表示应用最外层的组件

数据绑定常见的三种方式

{{}}：属于单向数据绑定

v-model：属于双向数据绑定

v-bind：属于单向数据绑定

方法一：插值表达式

{{}}, 里面可以写 js 表达式

支持运算

方法二：v-model

方法三：v-bind

和第一种方式差不多, 只是把 {{}} 换成了 v-bind, 用在标签的属性上

比如：给图片添加 title
