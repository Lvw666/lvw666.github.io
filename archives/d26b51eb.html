HTML 表单

发表于2022-04-22|更新于2022-04-22|Web 前端☛ HTML
|字数总计:3.2k|阅读时长:15分钟|阅读量:

表单是什么

表单用来收集信息，比如注册、登录、发送评论反馈、购买商品等等

表单的创建

所有 HTML 表单都以一个<form>元素开始

表单控件

单行文本框 input:text

1、使用 type 属性值被设置为 text 的<input>元素可以创建单行文本框，它是一个单标签

2、value 属性表示已经填写好的值

3、placeholder 属性表示提示文本，将以浅色文字写在文本框中，并不是文本框中的值

4、disabled 属性表示用户不能与元素交互，即锁死

密码框 input:password

1、使用 type 属性值被设置为 password 的<input>元素可以创建密码框

2、密码框里面输入的信息, 用户不可见

单选按钮 input:radio

1、使用 type 属性值被设置为 radio 的<input>元素可以创建单选按钮

2、互斥的单选按钮应该设置它们的 name 为相同值

3、单选按钮要有 value 属性值，向服务器提交的就是 value 值

4、传值的时候, 需要加上 name 属性, 标记是谁的值

5、如果是 get 请求, 提交到后台的数据的格式如下

6、单选按钮如果加上了 checked 属性，表示默认被选中

label 标签

1、label 标签用来将文字和单选按钮进行绑定，用户单击文字的时候也视为点击了单选按钮

2、在 HTML4 时代，label 标签是通过 for 属性和单选按钮的 id 属性进行绑定的

复选框 input:checkbox

1、使用 type 属性值被设置为 checkbox 的<input>元素可以创建复选框

2、同组复选框应该设置它们的 name 为相同值

3、复选框要有 value 属性值，向服务器提交的就是 value

4、复选按钮如果加上了 checked 属性，表示默认被选中

下拉菜单 select>option

1、<select>标签表示下拉菜单，<option>是它内部的选项

2、添加 selected 属性, 可以设置下拉菜单的默认选中项

多行文本框 textarea

<textarea></textarea>表示多行文本框

rows 和 cols 属性，用于定义多行文本框的行数和列数

三种按钮

表单中常见三种按钮，它们也都是 input 标签，type 属性值不同

button 和 input:button

都可以实现按钮的效果

button 和 input:button 区别：

1、input:button 会生成一个普通按钮, 如果在表单之中, 没有提交表单的功能

2、在 form 里面的 button, 是可以提交表单的

3、<button>按钮，可以设置 type 属性，当添加 type 属性时，它就不具备提交表单的能力了

4、button 因为是双标签, 所以可以添加图片, 不过默认会有一圈轮廓

5、带不带图片都会触发提交功能, form 里面的 button 自带提交功能

input:image

1、input 是单标签, 没有办法放图片, 除非使用 input:image, 渲染出来的图片按钮, 没有轮廓

2、注意: input:image可以提交表单, 同时会带上 x,y, 即鼠标点的位置

input:submit

1、<input type="submit"> 可以提交表单, 效果相当于<button></button>

input:reset

<input type="reset"> 可以重置表单, 就是所有的输入内容清空, 重新填写

因为会所有内容都清空, 所以用户体验不是很好, 不推荐使用

按钮总结

按钮一共有 7 种

如何选择使用哪一个按钮

由于页面布局具有多样性，实现方式并不唯一。当多个按钮都能实现效果时，那么选择哪一个都行。当然也可以根据实际情况选择，例如：

  1. 如果想要一个图片形式的按钮，那么优先考虑<button></button>按钮 <input type="image">
  2. 如果想利用 form 标签提交数据，那么按钮优先考虑<button></button>和 type 为 submit 的按钮
  3. 如果不想利用 form 标签提交数据，那么可以优先考虑选择 type 为 button 的按钮

input 类型总结

HTML5 新增控件

更丰富的 input 种类

color 控件

可以让用户选择颜色

date 控件

date 可以让用户快速的输入日期类型的数据

email 控件

可以检测用户输入的邮箱是否正确

file 控件

可以开一个弹框, 让用户选择文件上传, 常用于设置头像, 上传附件

number 控件

用户只能输入数字, 输入非数字字符, 无法写入 input 框

range 控件

会出现一个进度条, 可以快速选择进度

datalist 控件

datalist 控件可以为输入框提供一些备选项，当用户输入的内容与备选项文字相同时，将会显示智能感应

datalist 和 select的区别

外观不同

datalist 支持搜索, 下拉框不支持搜索

代码实现也不一样
