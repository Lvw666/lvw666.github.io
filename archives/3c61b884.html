什么是数组

编程时，我们经常会遇见处理一组数值的情况，比如全班 10 个学生的考试成绩

计算最大值, 最小值, 平均分会很不方便, 所以不推荐多个变量的方式

数组（Array），顾名思义，用来存储一组相关的值

从而方便进行求和、计算平均数、逐项遍历等操作。

每种高级编程语言中都有数组，它是非常重要的一种数据结构

数组的基本操作

数组如何定义

定义一个数组非常的简单，只需要使用方括号[]即可, 这种方式工作中最常用

数组里面的每个元素, 也叫数组的一项

数组和字符串一样, 也有长度

定义数组的另外一种方式

这是属于面向对象的一种定义方式

访问数组项

数组每一项都有下标，下标从 0 开始, 最大的下标, 是数组长度-1

可以使用方括号中书写下标的形式，访问数组的任一项

下标越界

JavaScript 规定，访问数组中不存在的项会返回 undefined，不会报错

数组长度

数组的 length 属性表示它的长度

数组最后一项的下标是数组的长度减 1

更改数组项

数组并不是只读的，我们可以修改它其中任何项的值

如果更改的数组项超过了 length-1，则会新增一项

数组的遍历

数组的最大的优点就是方便遍历

数组类型的检测

数组用 typeof 检测结果是 object

Array.isArray() 方法可以用来检测数组

数组常用方法

push()

push()方法用来在数组末尾添加新元素，参数就是要新增的元素

调用 push()方法后，数组会立即改变，不需要赋值

返回值是数组最终的一个长度

如果要新增多项，可以用逗号隔开

pop()

英语中是弹出的意思, 和 push 的操作正好相反, 删除末尾的一项, 末位淘汰

pop()方法不仅会删除数组末项，而且会返回被删除的项

unshift()

unshift()方法用来在数组开头添加一个新元素, 参数就是要添加的新元素

调用 unshift()方法后，数组会立即改变，不需要新变量来接收

返回值是数组最终的一个长度

如果要插入多项，可以用逗号隔开

shift()

与 unshift()相反，shift()方法用来删除数组中下标为0的项, 就是从开头删除一个, 淘汰销冠

shift()方法不仅会删除数组元素，而且会返回被删除的元素

splice()

替换数组中的指定项

从下标为 3 的项开始，连续替换 2 项

把 DE, 替换成 XYZ

会返回被替换掉的元素[‘D’,’E’]

在指定位置, 添加新元素, 只添加, 不删除

会返回被替换掉的元素[]

可以删除指定位置的元素

可以使用如下方法, 删除数组中的特定项

使用 for 循环, 配合 splice, 删除’F’, 代码可以这么写

配合 indexof, 还有一种简化的写法

如果想删除所有的”F”, 提供两种思路

slice()

slice() 方法用于得到子数组，类似于字符串的 slice() 方法

slice(a,b) 截取的子数组从下标为 a 的项开始，到下标为 b（但不包括下标为 b 的项）结束

slice(a,b)方法不会更改原有数组

slice()如果不提供第二个参数，则表示从指定项开始，提取所有后续所有项作为子数组

slice()方法的参数允许为负数，表示数组的倒数第几项

concat()

concat()方法可以合并连结多个数组, concat()方法不会改变原数组

reverse()

reverse()方法用来将一个数组中的全部项顺序置反, 会改变原来的数组

思考: 如何把”ABCDEFG” 变成 “GFEDCBA”?

split 字符串转数组

split 参数, 如果是空字符串, 每个字符拆成一个, 如果是分隔符, 按分隔符拆分字符串, 如果什么都不写, 所有字符当成一个整体, 拆成一个

数组拼接成字符串 join

join 参数是连接符, 通过连接符, 连接成字符串

字符串和数组互转

join 连接, split 分割

字符串和数组更多相关性

字符串也可以使用方括号内写下标的形式，访问某个字符，等价于 charAt()方法

indexOf()

indexOf()方法的功能是搜索数组中的元素，并返回它所在的位置，如果元素不存在，则返回-1

判断值是否存在时, 用的是===

根据索引, 删除数组中的某一项

冒泡排序

效率不高, 但是比较经典, 面试常考

从[30,22,19,13,35,8] 变成[8,13,19,22,30,35]

动图视频: https://www.bilibili.com/video/BV1f34y1s7ao

所谓冒泡排序, 就是比较左右两边的数的大小, 两种排序模式, 从左往右/从右往左

  1. 从左往右(下沉): 左i, 右i+1, i>i+1, 定义一个空变量tmp, 把左边大的i丢进去, 原本装大的 i+1位置 的容器arr[i]就空出来了, 再把右边小的i+1丢进去, 然后原本装i+1的容器arr[i+1]也空出来了, 再把开始装在tmp里面的i丢进arr[i+1]里, 相当于利用数组来完成一次变量的互换

  2. 从右往左(上浮): 同理, i<i+1, 定义一个空变量tmp, 把右边大的i+1丢进去, 装 i+1的容器arr[i+1]就空出来了, 再把左边小的i丢进arr[i+1]里, 最后再把tmp中的i+1丢进arr[i]中, 完成互换

注意: i 和 i+1 均代表数组中的项, 实际开发中要加上arr[]

练习: 已知数组为[6, 2, 9, 3, 1, 8, 5, 7], 使用冒泡排序, 输出[1, 2, 3, 5, 6, 7, 8, 9], 使用 js 实现

复习基本类型和引用类型

基本类型：number、boolean、string、undefined、null

引用类型：array、object、function、regexp(正则表达式)…

内存和硬盘的分工不同, 内存更快更贵, 主要是运行程序, 硬盘慢一点, 也更便宜, 用来保存文件

基本类型值变量传值的内存变化

引用类型值变量传值的内存变化

不同类型, 相等判断时有区别

基本类型进行相等判断时，会比较值是否相等

引用类型进行相等判断时，会比较址是否相等，也就是说它会比较是否为内存中的同一个东西

不要用 arr == [] 来判断空数组, 应该使用 length, 判断长度是否为 0

总结如下

举例 var a = b; 传值 a == b 判断
基本类型 数字, 字符串, 布尔… 内存中生成新副本 ``比较值`
引用类型 对象, 数组 不产生新副本, 新变量指向同一个对象 比较内存地址

深克隆和浅克隆

浅克隆

使用arr1=arr2的语法不会克隆数组, 因为指向同一个内存空间, 两个变量并没有分开

浅克隆：只克隆数组的第一层，如果是多维数组，或者数组中的项是其他引用类型值，则不克隆其他层, 藕断丝连
