JS 表达式与运算符

表达式和运算符

算数运算符

加减的符号和数学一致，乘法是*号，除法是/号

默认情况，乘除法的优先级要高于加法和减法；必要时可以使用圆括号来改变运算的顺序

加号的两种作用

加号有加法和连字符两种作用

如果加号两边的操作数都是数字，则为加法，否则为连字符

取余运算

取余运算也叫作求模运算，用百分号%表示

a % b表示求 a 除以 b 的余数，它不关心整数部分，只关心余数

被除数 = 商 × (除数) + 余数

隐式类型转换

如果参与数学运算的某操作数不是数字型，那么 JavaScript 会自动将此操作数转换为数字型

隐式转换的本质是内部调用 Number()函数

有关 IEEE754 toFixed()

在 JavaScript 中，有些小数的数学运算不是很精准

JavaScript 使用了 IEEE754 二进制浮点数算术标准

这会使一些个别的小数运算产生“丢失精度”问题, 因此不要进行小数的相等判断

解决办法：在进行小数运算时，要调用数字的 toFixed()方法保留指定的小数位数, 返回的是字符串

幂和开根号 Math.pow()

JavaScript 中没有提供幂计算、开根号的运算符

需要使用 Math 对象的相关方法进行计算

向上/向下取整 Math.ceil()

Math.ceil() 向上取整； Math.floor()向下取整

向上取整: 数字为正数, 只要数字后面有小数位, 就取该数字 +1 的整数（负数相反）

向下取整: 数字为正数, 不管数字后面的小数多大, 都取该数字自身的值（负数相反）

四舍五入 Math.round()

如果四舍五入, 使用Math.round(数字)

随机数 Math.random()

基础写法

Math.random(): 随机获取范围内的一个数（ 精确到小数点后14位 ）

生成指定范围内的随机数

  1. 生成 [0, m] 的随机数（含小数）
  1. 生成 [0, m] 的整数

可以使用 parseInt 去除小数点的形式将生成的随机数转换为整数

  1. 生成向下取整的随机整数

使用Math方法下的floor属性进行舍弃小数向下取整, 当然你也可以使用 Math.ceil 向上取整

生成两个数之间的随机数

  1. 生成 [n, n+m] 之间的随机数
  1. 生成 [-n, n+m] 之间的随机数
  1. 生成 [-10, 0] 之间的随机数
  1. 生成 [n, m] 之间的随机整数（包括n与m）

关系运算符

关系运算符, 返回的是一个布尔值, 要么是 true, 要么是 false

大于和小于

大于>、小于<两个符号和数学相同

“大于等于”运算符是>=，“小于等于”运算符是<=

JS中没有连比

判断变量 a 是不是介于 3 到 15 之间，应该怎么写呢？

判断是否相等

如果想比较两个值是否相等，此时应该使用==运算符

JavaScript 中等号=表示赋值，而并不是相等

判断相等应该用==或者===

=== 和 == 的区别

== 运算符不比较值的类型, 它会进行隐式转换后, 比较值是否相等

=== 不仅比较值是否相同, 也比较类型是否相同

=== 更加严格

什么时候用==, 什么时候用===?

除了判断undefined和null的时候用==, 其他都用===

a == undefined 相当于 a === undefined || a === null

两等的结果, 供参考, 稍作了解

NaN 不自等

NaN 作为一个特殊的数字类型值，NaN 和 NaN 不相等

如何判断某变量值为 NaN

isNaN()函数可以用来判断变量值是否为 NaN

但 isNaN()也不好用，它的机理是：

只要该变量传入 Number()的执行结果是 NaN，则 isNaN()函数都会得到 true

特殊值的相等关系

不相等和不全等

!=表示不相等，!==表示不全等

逻辑运算符

非运算

!表示“非”，也可以称为“取反运算”

!是一个“单目运算符”，只需要一个操作数

取反运算的结果一定是布尔值

工作中有时会用到 !! 来转布尔值

与运算

&&表示“并且”，称为“与运算”，口诀：都真才真

必须同时满足所有条件, 比如高&&富&&帅&&男, 高富帅

或运算

||表示“或者”，称为“或运算”，口诀：有真就真

条件很多, 满足一个即可, 比如老||弱||病||残||孕专座

短路运算

请思考下面表达式的运算结果

a && b运算中：a 真，表达式值为 b；a 假，表达式值为 a

a || b运算中：a 真，表达式值为 a；a 假，表达式值为 b

适用于赋初始值

以下结果, 也是遵循短路计算的

逻辑运算顺序

逻辑运算的优先级是：非 ! → 与 && → 或 ||

赋值运算符

JS 中， =表示赋值， ==判断是否相等（不判断类型）,===判断是否全等

赋值运算符会将等号右边的数值，赋予等号左边的变量

赋值运算也产生值

赋值运算也产生值，等号后面的值将作为“赋值运算的值”

这就意味着，可以连续使用赋值运算符

快捷赋值

快捷赋值运算符表示在原数值基础上进一步进行运算

自增/自减运算符

++和--是自增/自减运算符，表示在自身基础上加 1 或减 1

a++ 和 ++a 的区别

a++ 先用再加； ++a 先加再用

综合运算的运算顺序

运算顺序： 非运算 → 数学运算 → 关系运算 → 逻辑运算
