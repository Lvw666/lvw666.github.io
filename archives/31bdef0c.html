CSS 布局概述

发表于2022-04-22|更新于2022-04-22|Web 前端☛ CSS

什么是css布局

定义网页各个元素的排列方式, 把元素按正确的大小, 摆放在正确的位置上

传统PC端布局的三种方式

  1. 普通流, 标准流(各种标签, 按照默认方式进行排列)
  2. 浮动
  3. 定位

实际开发过程中, 需要三种方式, 配合使用

什么是标准流

标准流：

又称文档流，是浏览器在渲染显示网页内容时默认采用的一套排版规则，规定了应该以何种方式排列元素

常见标准流排版规则：

  1. 块级元素：从上往下，垂直布局，独占一行 (从上往下)
  2. 行内元素 或 行内块元素：从左往右，水平布局，空间不够自动折行 (从左往右)

行内元素和块级元素的区别

display 默认宽度 默认高度 可设置宽高 是否独占一行
block 父元素宽度 内容高度 是 是
inline 内容宽度 内容高度 否 否

设置元素为块级元素

display:block, 可以设置元素为块级元素, 独占一行

可以设置宽高, 不设置的话, 宽度相当于父级 100%, 高度默认由内容撑开

典型的块级元素标签: div、p、h、ul、li、dl、dd、form、header、section、table、nav、footer

设置元素为行内元素

display:inline, 可以设置元素为行内元素, 可以在一行中显示, 如果超过宽度, 会自动换行

不可以设置宽高, 宽度高度由内容撑开

典型的行内元素: a、span、b、u、i、s、strong、ins、em、del、img、input

设置元素为行内块元素

display:inline-block, 可以设置元素为内联块级元素

一行可以显示多个, 可以设置宽高

拥有inline-block特征的标签, input、textarea、button、select、img

行内元素和块级元素相互转换

使用 display: block; 即可将元素转为块级元素

使用 display: inline; 即可将元素转为行内元素，将元素转为行内元素的应用不多见

使用 display: inline-block; 即可将元素转为行内块

设置元素的显示和隐藏

第一种隐藏的方式

可以通过display: none;属性控制显示和隐藏

第二种隐藏的方式

visibility: hidden;, 也可以隐藏元素

两种隐藏方式的区别

都可以隐藏元素

display:none 会让元素消失, 元素原来的位置会空出来, 如果需要显示, display:block

visibility:hidden, 会让元素消失, 但是位置保留, 相当于隐身了, 如果需要显示visibility: visible

visibility : 能见度

hidden, hide: 隐藏

第三种隐藏的方式

opacity: 0;设置透明度也可以起到隐藏的效果, 相当于visibility: hidden;

HTML标签嵌套规范

块级元素一般作为大容器, 可以嵌套: 文本, 块级元素, 行内元素, 行内块元素

注意: p标签不要嵌套div,p,h块级元素。

a标签内部可以嵌套任意元素

注意: a标签里面不能再次嵌套a标签。
