switch 语句

switch 分支流程控制

switch 语句也是多分支语句，它用于基于不同的条件来执行不同的代码。

针对变量设置一系列的特定值的选项时，就可以使用 switch。

case 穿透特性

在 switch 语句中，如果 case 的后面不写 break，将出现 case 穿透现象

break 的作用, 终止语句体代码的执行, 跳出 switch

case 穿透不是 bug, 是特性, 根据需要合理使用, 可以简化代码

循环

for 循环

任务: 输出一千遍”老婆我错了”

for 的圆括号中有三个表达式:

  1. 表达式var i = 0; 表示定义一个“循环变量”i，并赋值为 0；
  2. 表达式i < 1000; 表示继续执行循环的条件，只要这个条件为真，则会一直执行；
  3. 表达式i++用来更新循环变量，使循环变量的值越来越趋向终点。

执行顺序: 1 -> 2 -> 3 -> 4 -> 2 -> 3 -> 4 -> 2 -> 3 -> 4 …

  1. 首先会执行语句 ①
  2. 然后判断语句 ② 是否满足，如果满足则进入循环体，执行语句 ③，如果不满足则退出循环
  3. 语句体中的语句执行完毕后，执行语句 ④
  4. 再次判断语句 ② 是否满足，如果满足则进入循环体，执行语句 ③，如果不满足则退出循环
  5. ……

while 循环

while 语句事先不指定循环开始、结束的范围，只要测试条件满足，就一直执行循环体

如果测试条件一直是真, 就会形成一个无限循环

死循环和无限循环的区别在于, 死循环是意料之外, 无限循环是意料之内

while 循环实现累加

计算 1+2+3+…+100, 使用 while 循环

while 和 for 的区别

第一, 写法不一样, 第二, 使用场景不一样

for 适用于次数明确的循环, 500 个俯卧撑

while 适用于条件明确的循环, 铺张报纸, 汗水湿透

do while 循环

do while 循环是一种“后测试循环语句”。

它不同于 for 循环和 while 循环每次都是“先测试条件是否满足，然后执行循环体”

do-while 循环是“先执行循环体，然后测试条件是否满足”。

do-while 循环将循环执行条件写到了循环体的后面，这样一来，循环体一定会至少执行一次，然后再检测循环执行条件是否为 true，决定是否继续执行循环体。

while 相当于前门保安, do…while 相当于后门保安

break

break(中断) 表示立即终止循环，用在循环语句中，在 for 循环和 while 循环中都可以使用

break 用在 while 语句中，通常和while(true){}搭配使用

比如: 计算平方大于 12345 的最小数字

continue

continue(继续) 用于跳过循环中的一个迭代，并继续执行循环 中的下一个迭代。

for 循环更经常使用 continue, 比如逢七过
