MongoDB 下载安装

MongoDB 官网

分为服务端和客户端

  1. 服务端下载安装

安装步骤如下

自定义选择合适安装位置

取消勾选安装 MongoDB Compass

点击安装完成即可

在 data 中创建 db 文件夹

在 bin 目录下使用如下命令创建服务（因为其他目录下, 没有mongod.exe, 会导致命令执行失败）

在输出的信息中, 可以看到对应的 ip 和端口

  1. Mongodb 的默认端口号, 是 27017
  2. windows 中, Mongodb的服务开启一次即可, 后面重启电脑, 服务依旧存在
