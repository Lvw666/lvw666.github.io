scoped 的注意事项

  1. 使用 scoped 后，父组件的样式将不会渗透到子组件中
  2. 不过一个子组件的根节点会同时受其父组件的 scoped CSS 和子组件的 scoped CSS 的影响
  3. 这样设计是为了让父组件可以从布局的角度出发，调整其子组件根元素的样式
